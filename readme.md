# Description

Hash Checker is a TUI Program to check hashes of files.

# Features
Auto-detect Hash based on length

```
##############################
# HASH       | Chars Long    #    
#############|################
# MD5        |  32           #
# SHA1       |  40           #
# SHA224     |  56           #
# SHA256     |  64           #
# SHA384     |  96           #
# SHA512     |  128          #
##############################
```
